mindmap
  root((Карьерный путь frontend-разработчика (4 года)))
    Стартовые условия
      Возраст 20–21, базовые навыки ПК
      Нулевые/базовые знания HTML, CSS, JS
      2–3 часа в день на обучение
      ПК + интернет + GitHub
      Цель: первая работа junior за 12–18 месяцев
    Конечный результат к 24–25 годам
      Должность Middle/Senior Frontend Engineer
      Стек: HTML, CSS, JS, React/Vue, TypeScript
      Понимание адаптивности, accessibility, тестирования
      Опыт коммерческих проектов и pet-проектов
      Участие в product/архитектурных решениях
    Поэтапный план развития
      Этап 1 (0–12 месяцев)
        HTML5, семантика, формы, таблицы
        CSS3, Flexbox, Grid, адаптив
        Базовый JavaScript: типы, функции, DOM, fetch
        Инструменты: Git, npm, DevTools, сборщик
        3–5 учебных проектов, выкладка на GitHub Pages/Netlify
        Стажировка или первая позиция trainee/junior
      Этап 2 (12–24 месяцев)
        Основной фреймворк (React/Vue): компоненты, хуки/реактивность
        TypeScript: типы, интерфейсы, generics
        State-management (Redux/Pinia/Zustand и др.)
        Качество кода: ESLint, Prettier, тесты, code review
        Работа в команде, Scrum/Kanban, фичи и багфиксы
        Уверенный junior / начинающий middle
      Этап 3 (24–48 месяцев)
        Производительность, оптимизация, SSR/SSG (Next/Nuxt)
        Расширение стека: Node.js, REST/GraphQL, базовый backend
        CI/CD, Docker, деплой в облако
        Проектирование фронтенда, ревью, менторство
        Внешняя активность: статьи, выступления, open-source
        Роль strong middle / senior frontend
